# Java Basic Lessons

This folder contains Java code samples from introduction to methods.

## How to Run

Open Terminal and go to this project:

```bash
cd /workspace/java-basic-lessons/src
```

Compile the code:

```bash
javac Main.java
```

Run the program:

```bash
java Main
```

## Command Meaning

| Command | Meaning |
|---|---|
| `cd /workspace/java-basic-lessons/src` | Go to the folder that has `Main.java` |
| `javac Main.java` | Compile Java source code into `.class` files |
| `java Main` | Run the `Main` class |
| `ls` | Show files in the current folder |
| `pwd` | Show current folder path |
| `rm *.class` | Delete compiled class files if you want to clean |

| Class | Topic |
|---|---|
| `JavaIntro` | What Java is |
| `JavaOutput` | `System.out.println` and `System.out.print` |
| `JavaComments` | Single-line and multi-line comments |
| `JavaVariables` | Store data in variables |
| `JavaDataTypes` | `int`, `double`, `char`, `boolean`, `String` |
| `JavaTypeCasting` | Convert one data type to another |
| `JavaOperators` | Math and comparison operators |
| `JavaStrings` | String methods |
| `JavaMath` | Java `Math` helper methods |
| `JavaBooleans` | `true` and `false` values |
| `JavaIfElse` | Conditional logic |
| `JavaSwitch` | Choose between many cases |
| `JavaWhileLoop` | Repeat while condition is true |
| `JavaForLoop` | Repeat with a counter |
| `JavaBreakContinue` | Stop or skip loop steps |
| `JavaArrays` | Store many values in one variable |
| `JavaMethods` | Reusable code blocks |
| `JavaMethodParameters` | Send values into methods |
| `JavaMethodOverloading` | Same method name with different parameters |
| `JavaScope` | Where variables can be used |
| `JavaRecursion` | A method that calls itself |

## Important Rule

Because the code has:

```java
public class Main
```

the file name must be:

```text
Main.java
```

If the file name is different, Java will show an error.
